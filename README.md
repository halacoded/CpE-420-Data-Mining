# Project 1 – Classification

## Overview

This project applies supervised machine learning classification techniques on:

- Census Income Dataset
- Credit Approval Dataset

## Algorithms Implemented

- Decision Tree (DT)
- Random Forest (RF)
- Naive Bayes (NB)

## Main Tasks

- Dataset preprocessing
- Training and testing dataset generation (80/20 split)
- Random sampling without replacement
- Model training and classification
- Accuracy, precision, and recall evaluation
- Runtime comparison
- Handling missing data techniques

## Files Included

- `DTvsRFvsNB.py`
- `Project1Report-[lastname].pdf`
- Training/Test datasets
- `README.txt`

# Project 2 – Clustering

## Overview

This project focuses on unsupervised learning and text clustering using the:

- Asian Religious and Biblical Texts Dataset

## Algorithms Implemented

- K-Means Clustering
- K-Medoids Clustering

## Main Tasks

- Cluster analysis using multiple k values
- Silhouette Coefficient evaluation
- Clustering quality comparison
- Runtime analysis
- Text clustering observations and discussion

## Files Included

- `cluster_analysis.py`
- `Project2Report-[lastname].pdf`
